# ModernClothes

Sistema de tienda online en terminal para una tienda de ropa

## Descripción

Permite a los clientes navegar por un catálogo de ropa, agregar productos a su carrito y realizar compras

## Características

- Catálogo de productos dividido en categorías
  - **Camisas/Camisetas**: 5 opciones (Tank Top, T-Shirt, Polo, Dress Shirt, Jacket)
  - **Shorts/Pantalones**: 5 opciones (Bermuda, Running, Cargo, Denim, Baggy)

- Carrito de compras persistente
- Cálculo automático de factura
- Formulario de datos del cliente (nombre, teléfono, dirección)
- Verificación y corrección de datos antes de confirmar
- Navegación intuitiva con menús seleccionables
